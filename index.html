My Framework Page: MaterializeCSS

How I Installed the Framework

I used MaterializeCSS because it is a simple front-end CSS framework. To include it, I just added the CDN link in the <head> section of my HTML file: 

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css">
How I Built the Page

I created this HTML page in GitHub using a text editor. I used Materialize classes for containers, rows, columns, and cards to structure the page.

Each section is placed inside a <div class="card"> to make it visually organized.

Difficulties and Solutions
  • Initially, I tried Bootstrap, which is not allowed, so I switched to MaterializeCSS.
  • At first, the page didn’t show on GitHub Pages. I fixed it by creating index.html at the root of the repository and enabling Pages from the main branch.
  • I also had to wait a few minutes for GitHub Pages to publish the changes.

This page is hosted publicly on GitHub Pages and demonstrates my use of a front-end framework.
